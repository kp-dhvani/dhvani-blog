the only four calls that matter

// 1. the resting string is just a flat 2-point "curve"
const string = new Bezier(x0,y, (x0+x1)/2,y, x1,y);

// 2. where on the string did the user click? -> nearest on-curve point + its t
const hit = string.project({x:mx, y:my});   // {x, y, t, d}

// 3. fret it: divide into the sounding segment + the muted finger side
const {left, right} = string.split(hit.t);

// 4. the monochord ratio falls straight out of the segment lengths
const ratio = string.length() / left.length();
const freq  = fundamental * ratio;          // pitch = f0 · fullLen / soundingLen

// rendering: sample whichever curve you want, draw your own polyline
left.getLUT(48).forEach(p => ctx.lineTo(p.x, p.y));
